Welcome to TBI’s documentation!

TBI [1] is a programming language written in Python 3.12, and is designed to limit the programmer’s accessibility to common high-level features, some including variables and functions, forcing the developer to think more about how to preform common tasks with limited memory and resources.

You can download TBI here.

Note

These python modules are required by TBI to function:

  • os (built-in)

  • random (built-in)

  • time (built-in)

  • py-term

  • termcolor

  • argparse

You can install them with:
pip install py-term termcolor argparse

Note

This project is under active development. Expect things to change.
